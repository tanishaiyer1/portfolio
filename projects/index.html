Formula 1 Sector Analysis (2018–2021)

Research Question: Do track sectors with the highest number of corners show the strongest correlation with final qualifying positions?

Contributed to a group research project analyzing Formula 1 qualifying data (2018–2021) using the FastF1 API. We explored whether corner-dense sectors exerted the greatest influence on qualifying outcomes, applying regression analysis to compare sector complexity against final grid positions.

  • Helped extract and clean multi-year race timing data from the FastF1 API.
  • Worked on exploratory data analysis and regression modeling.
  • Assisted in presenting findings showing that corner-dense sectors had the strongest correlation with grid outcomes.

Tools: Python (pandas, matplotlib), FastF1 API, regression analysis
